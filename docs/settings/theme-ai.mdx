# Theme AI — Appearance

Use Theme AI to generate on‑brand themes safely.

Prerequisites
- Feature flag: `FEATURE_FLAG_THEME_AI=1`
- Roles: Super Admin or Admin

Steps
1. Go to Settings → Appearance → Theme AI.
2. Click Generate and describe your theme (no secrets or PII).
3. Review the preview diff of tokens.
4. Click Apply to save a snapshot.
5. Rollback: pick any prior snapshot.

Troubleshooting
- If Generate is blocked, remove secrets/PII from your prompt.
- Ensure you have Admin/Super Admin role.
- Rate limits may apply.
